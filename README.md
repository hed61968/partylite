## 一个从Cape Cod厨房起家的蜡烛品牌

故事得从1909年说起。那时候有位叫Mabel Baker的女教师,在自己Cape Cod的小厨房里,用野生浆果给蜡烛上色、添香。你能想象那个画面吗?一个普通的美国主妇,在炉灶边捣鼓着蜡烛,结果做出了新英格兰地区最好的蜡烛。

到了1973年,这个小作坊正式成立了PartyLite Gifts公司。他们搞了个当时挺新鲜的销售模式——家庭派对式销售。就是你邀请朋友来家里,一边聊天喝茶,一边试用各种蜡烛。这种模式火得一塌糊涂,很快就推广到全球20多个国家。

现在PartyLite已经是Luminex Home Décor & Fragrance旗下的品牌,每年生产接近2亿支蜡烛。但他们始终坚持一点:用最干净的原料,做最亮的蜡烛。
